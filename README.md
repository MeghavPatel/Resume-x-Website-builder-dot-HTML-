# Browser-Based Resume / Portfolio Generator

A lightweight, static, browser-based tool that generates a personal
resume/portfolio website based on user inputs.

No backend. No database. Just open the HTML file and build a website.

## Features
- Input name, profile photo, interests, goals, projects, achievements
- Automatically generates a clean portfolio layout
- Fully browser-based (works offline)
- Customizable and beginner-friendly
- Mobile and desktop compatible

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## Use Case
Originally built to help a UI/UX student quickly generate a
personal portfolio website without complex setup.

## How to Use
1. Clone or download the repository
2. Open `index.html` in any modern browser
3. Fill in the form
4. Get your portfolio website instantly
